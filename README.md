# E-Commerce Analytics Platform

An end-to-end analytics engineering project built in Databricks using Delta Lake and Medallion Architecture principles.

## Technologies

- Databricks
- PySpark
- Delta Lake
- Azure Data Lake Storage
- SQL

## Architecture

Raw CSV Files
↓
Bronze Layer
↓
Silver Layer
↓
Gold Layer
↓
Business Analytics
